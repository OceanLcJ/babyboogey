## Seedance 宝宝跳舞 Prompt 公式

推荐结构：

```text
[主体] + [舞蹈风格] + [场景] + [表情] + [动作质量] + [安全/风格要求]
```

示例：

```text
Adorable baby doing a gentle boogie dance, bright smile, cozy nursery background, soft lighting, smooth realistic motion, family-friendly, short vertical video.
```

一个好 prompt 通常包含：

- 主体：baby、toddler、infant、little kid。
- 动作：boogie、hip-hop、disco、salsa、waltz、cute wiggle。
- 情绪：happy、playful、sweet、funny、wholesome。
- 场景：nursery、party room、living room、holiday scene。
- 质量：smooth motion、natural face、stable body、HD。
- 格式：short clip、vertical video、social-ready。

## 常见设置建议

如果工具支持参数控制，可以先这样设置：

- 时长：第一轮用 5 到 8 秒。
- 画幅：TikTok、Reels、Shorts 用 9:16。
- 动作强度：大头照用低或中等。
- 画质：有 HD 就选 HD。
- 镜头：稳定镜头或轻微运动。

## 常见错误

### Prompt 太复杂

一次要求舞蹈、服装、复杂镜头、背景变化、特效和多角色，很容易让结果不稳定。先保持一个主体、一个动作、一个场景。

### 动作和照片不匹配

大头照不适合极强的全身动作。近景照片用温和动作，上半身或全身照片再尝试更强的舞蹈。

### 写入隐私信息

不要在 prompt 里写真实姓名、地址、学校名或其他私人信息。Prompt 只描述风格和动作即可。

### 和照片状态冲突

如果照片里的宝宝在睡觉，要求极限街舞通常不自然。让 prompt 的能量和照片本身匹配。
